# Hello :)
This is the simple demo react-native application

# Installation

`git clone git@github.com:byabuzyak/mob_test.git`

`cd mob_test && yarn install`

`cd ./ios && pod install`

# Run on simulator
`react-native start`

**Android:** `react-native run-android`

**iOS:** `react-native run-ios`

# Run tests
`cd mob_test && yarn jest`
